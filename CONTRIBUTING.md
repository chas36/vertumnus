# Contributing

## Как вносить изменения

1. Создать ветку от `main`
2. Сделать небольшое законченное изменение
3. Прогнать тесты
4. Открыть `Pull Request` с коротким описанием

## Перед PR

Рекомендуемый минимум:

```bash
python -m unittest discover -s tests -v
python -m compileall main.py core models ui tests
```

## Что важно для этого проекта

- Не коммитить виртуальные окружения, `build/`, `dist/` и кэш
- Не ломать профиль `Для проектора`
- Не ухудшать совместимость выходного `MP4`
- Для UI-изменений проверять обе темы: тёмную и светлую
- Для работы с дорожками учитывать, что проектор не умеет переключать их во время воспроизведения

## Стиль изменений

- Предпочтительны небольшие PR
- Если меняется поведение конвертации, желательно добавить или обновить тест
- Если меняется интерфейс, приложить скриншот

## Обсуждение изменений

Для багов и идей используйте issue-шаблоны GitHub.
